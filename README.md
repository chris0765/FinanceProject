# FinanceProject
2023-1 지능형정보컴퓨팅특론 프로젝트

## 개요
1. 네이버 주식 뉴스를 크롤링하여 종목 관련 뉴스 데이터를 수집
2. 수집한 뉴스의 문맥을 NLTK를 응용하여 분석
3. 특정 단어의 등장 여부와 주식의 상/하향 분석
4. 3번의 모델을 통해 특정 단어 등장 시 주식의 가격 변화 예측하는 프로젝트
